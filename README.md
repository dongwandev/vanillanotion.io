## 목표

- Vanilla JavaScript를 이용하여 Notion타입의 노트앱을 구현.

<aside>
💡

### 구현할 기능

- 해시 라우팅과 주소 동기화
    - 문서를 탐색하고 활성 문서의 제목과 본문을 편집하면, 자동저장과 디바운스가 상태 저장소에 반영
    - 문서 이동과 화면 동기화
- 서식 툴바
    - 제목 레벨, 목록, 코드 블럭, 인용구, 체크박스, 각종 DOM삽입.
- 즐겨찾기
    - 정렬된 리스트를 한번에 볼수 있음.
    - 별 토글로 즉시 반영
- 휴지통 팝오버
    - 삭제, 복원, 영구 삭제
- 빠른검색 모달
    - 입력과 동시에 필터링
    - 화살표와 엔터로 즉시 이동
    - 원하는 문서에 빠르게 접근하는것이 목표
- 설정 모달
    - 설정값 로컬에 영구 저장
        - JSON파일에 저장하여 설정 불러오기 내보내기 가능
    - 테마 변경
- 반응형 사이드바
    - 버튼 토글
    - 드래그 리사이즈
    - 더블클릭 스냅
    - 뷰포트 전환과 회전에 따른 반응형 제어
- 단축키
    - 검색 열기
    - 새문서
    - 인라인 이름 바꾸기
- 문서 관리
    - 중첩문서 관리
        - 하위페이지
